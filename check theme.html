Terceirizar logotipo

Apenas se o cliente não possuir.

Criar paleta de cores

Criar paleta de cores com logotipo e/ou segmento da empresa.

Desenvolver protótipo

Desenvolver protótipo do site em servidor local, depois apresentar para o cliente.

Imagens e textos

Pegar fotos com o cliente ou banco de imagens e os conteúdos textuais do site. Contratar copywriter caso necessário.

Criar senha

Elaborar uma senha e nome de usuário para o cliente com letras, símbolos e números.

Criar conta Hostgator

Criar uma conta hostgator, exportar site criado, comprar domínio e hospedagem.

Criar emails

Criar 2 emails profissionais para o cliente e fazer direcionamento para o gmail principal.

Plugins e configurações

Instalar plugins [Yoast SEO, Contact Form] e colocar configurações básicas para o WordPress [SEO, SSL, Responsividade, usabilidade, carregamento, compatibilidade com vários navegadores, Google lite house, Google mobile friendly, LGPD, atualização automática do WordPress, Google Analytics, Google Search Console, botão do WhatsApp].

Redes sociais

Criar IG e/ou FB caso o cliente não tenha. Acrescentar linktree ou cartão de visitas digital, duas postagens e uma imagem de perfil com a logo do website sem cobrar nada.

Google meu negócio

Configurar palavras-chave, descrição, as imagens do estabelecimento (se possível), os dados da empresa, o email profissional e o endereço do site.

Entregar site para o cliente

Logotipo em PDF e PNG, passar o endereço do site, a conta da Hostgator, o usuário, a senha, ensinar o básico do WordPress para o cliente em vídeo ou reunião, caso ele não conheça. Avisar sobre tarifa anual do domínio e hospedagem via boleto, o valor que será cobrado.
